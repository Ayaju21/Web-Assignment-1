# Web Development Assignment – HTML Website Project



## 📌 Overview
This project is a university web development assignment focused on building a structured HTML website. The assignment includes creating a personal homepage, organizing a proper website folder structure, and developing a simple clothing e-store website using semantic HTML.

---

## 🧩 Project Structure

The project is divided into three main tasks:

### 1️⃣ Folder Structure
A structured directory was created inside `public_html`, including:
- `ass/ass1/` → Assignment 1 files
- `examples/htmlEx/` → HTML examples
- `project/proj1/` → Project files

Each folder contains an `index.html` file for navigation and organization.

---

### 2️⃣ Personal Home Page
A personal website page including:
- Personal information (name, ID, department, photo, etc.)
- Education history
- Training courses
- Internship / work experience
- Volunteer activities
- Awards and achievements
- Study timetable
- Navigation links to assignments and projects

The page uses semantic HTML elements such as:
`<header>`, `<nav>`, `<section>`, `<article>`, `<figure>`, and `<footer>`.

---

### 3️⃣ Clothing E-Store Website
A simple e-commerce website including:

#### 🏠 Home Page
- Store introduction
- Images and descriptions
- Saved as `ass1/index.html`

#### 🛍️ Product Pages
- Product name, image, price, description
- Customer reviews and ratings

#### 📞 Contact Page
- Address, phone, email
- Contact form (name, email, city, subject, message)

#### 🧾 Registration Page
- Customer registration form
- Uses POST method to submit data to:
  `http://comp334.studentswebprojects.ritaj.ps/util/process.php`

---

## ⚙️ General Requirements
- Semantic HTML5 structure
- Relative paths for all links
- Navigation bar and footer on all pages
- Footer includes last update, contact info, and support link
- HTML validation using W3C Validator

---

## 🎯 Objective
The goal of this assignment is to practice building structured, well-organized, and semantic HTML websites while following real-world web development standards.
